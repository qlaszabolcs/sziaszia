# Algorithms Seminar

Welcome to the Algorithms Seminar.

## Workflow Rules

1. Pull latest changes before starting.
2. Create a feature branch per assignment:
   feature/weekXX-topic
3. Commit frequently.
4. Open Pull Request before merging.
5. CI must pass before deadline.

## Submission Policy

The version in `main` at the deadline counts.

## Commit Message Format

feat: implement merge sort
fix: correct boundary condition
refactor: simplify recursion
test: add edge case

## Academic Integrity

- Do not copy solutions.
- Your commit history must reflect your development process.
- Large single commits will reduce grade.
